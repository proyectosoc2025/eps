HOSPITAL EPS - INSTRUCCIONES RÁPIDAS
========================================

PASO 1: COPIAR ARCHIVOS
------------------------
1. Copie toda la carpeta "hospitalEPS" a: C:\wamp64\www\
2. Ruta final: C:\wamp64\www\hospitalEPS\

PASO 2: CREAR BASE DE DATOS
----------------------------
1. Inicie WAMP Server
2. Abra: http://localhost/phpmyadmin
3. Cree una nueva base de datos llamada: EPS
4. Importe el archivo: database/eps_database.sql

PASO 3: AGREGAR IMAGEN
-----------------------
1. Descargue una imagen de hospital
2. Guárdela como: assets/images/hospital.jpg
3. Tamaño recomendado: 1920x1080px

PASO 4: ACCEDER AL SISTEMA
---------------------------
URL: http://localhost/hospitalEPS

========================================
FUNCIONALIDADES POR ROL
========================================

ADMINISTRADOR:
- Ver todos los datos del sistema
- Gestionar pacientes, doctores y administradores
- Crear/modificar/eliminar historias clínicas
- Ver logs del sistema
- Acceso completo

DOCTOR:
- Ver y editar su perfil
- Registrar pacientes
- Crear/modificar/eliminar sus historias clínicas
- Ver solo sus pacientes
- No puede ver datos de otros doctores

PACIENTE:
- Ver su perfil
- Ver sus historias clínicas
- Ver sus visitas médicas
- Modificar: contraseña, dirección, correo, teléfono
- No puede modificar otros datos
