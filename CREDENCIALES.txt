========================================
TABLAS DE LA BASE DE DATOS
========================================

1. administradores
   - Almacena usuarios administradores
   - Campos: id, nombres, apellidos, identificacion, usuario, 
     password, rol, cargo, edad, genero, direccion, telefono

2. doctores
   - Almacena información de doctores
   - Campos: id, nombres, apellidos, identificacion, usuario, 
     password, telefono, rol, direccion, profesion, edad, 
     genero, titulo_profesional, cargo

3. pacientes
   - Almacena información de pacientes
   - Campos: id, nombre, primer_apellido, segundo_apellido, 
     identificacion, usuario, password, edad, telefono, 
     direccion, profesion, raza, tipo_piel, sexo, 
     grupo_sanguineo, estatura, peso, correo_electronico, rol

4. historias_clinicas
   - Almacena historias médicas
   - Campos: id, paciente_id, doctor_id, admin_id, 
     fecha_consulta, motivo_consulta, diagnostico, 
     tratamiento, medicamentos, observaciones, sintomas, 
     examenes_realizados

5. visitas_medicas
   - Almacena registro de visitas
   - Campos: id, paciente_id, doctor_id, historia_clinica_id, 
     fecha_visita, motivo, observaciones

6. logs
   - Almacena logs de actividad
   - Campos: id, usuario, accion, descripcion, tabla_afectada, 
     registro_id, ip_address, fecha

========================================
FUNCIONES DE SEGURIDAD
========================================

detectSQLInjection($input)
- Detecta patrones de inyección SQL
- Registra intentos sospechosos en logs
- Bloquea el acceso

sanitizeInput($data)
- Limpia y sanitiza entradas de usuario
- Previene XSS
- Elimina caracteres peligrosos

logSuspiciousActivity($usuario, $ip, $descripcion)
- Registra actividades sospechosas
- Almacena IP del atacante
- Guarda timestamp del intento

========================================
MENSAJES DE SEGURIDAD
========================================

Intento de Inyección SQL:
"Usted está intentando realizar una operación sospechosa 
prohibida. Intento de acceso bloqueado."

Este mensaje se muestra cuando:
- Se detectan patrones SQL en inputs
- Se intentan caracteres especiales maliciosos
- Se detectan intentos de bypass de autenticación
